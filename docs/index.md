# PocketBase KMP Docs

PocketBase KMP is a Kotlin Multiplatform SDK with modular architecture.

## Modules

- `pocketbase-core`: result and shared models
- `pocketbase-client`: HTTP and admin services
- `pocketbase-auth`: auth endpoints and flows
- `pocketbase-records`: records CRUD and listing
- `pocketbase-files`: file token and URL helpers
- `pocketbase-realtime`: realtime subscribe/unsubscribe

Use the left navigation to jump to setup, guides, and API pages.
